## Project Title : Special Instructions for OOP (Paruj) Part ##

## Project Overview ##
it was a program similar to grab or lineman. It can ordering the order to the driver and also showing
the status also the driver detail such as name or vehicle

## Features ##
There are 4 classes
- First one is Person which is a Parent class of other 2 classes. It have the method the introduce
who you are.
- Customer class is a child class of Person. It keep the detail of customer. It has method that
  place the order to the driver
- Driver class is also child class of Person. It keep driver detail. The method is deliver show
  that the driver is delivering your item
- Last one is DeliveryOrder it keep the order detail and summary the final detail

## Structure ##
Like I show in feature topic. This program have many method separate by class
1.Person 
- introduce -> inttroduce who you are by name
2.Customer
- place_order -> exist the item
3.Driver
- deliver -> show the detail of order
